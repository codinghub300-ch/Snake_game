# Snake Game 

A classic Snake Game built with Python using Turtle Graphics and Pygame for sound effects and background music.

---

##  Features

- Multiple difficulty levels:
  - Easy
  - Normal
  - Hard
- Smooth snake movement
- Real-time score tracking
- High score system
- Pause functionality
- Sound effects and background music
- Random food generation
- Colorful snake body segments
- Keyboard controls
- Collision detection system

---

##  Technologies Used

- Python
- Turtle Graphics
- Pygame
- Random Module
- Threading

## Controls
| Key   | Action         |
| ----- | -------------- |
| W / ↑ | Move Up        |
| S / ↓ | Move Down      |
| A / ← | Move Left      |
| D / → | Move Right     |
| P     | Pause / Resume |
| 1     | Easy Mode      |
| 2     | Normal Mode    |
| 3     | Hard Mode      |

## Game Rules
Eat food to increase your score.
Avoid hitting the walls.
Avoid colliding with the snake’s body.
The game resets when a collision occurs.
